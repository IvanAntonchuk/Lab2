# Лабораторна робота №2
## Автоматизоване UI-тестування веб-застосунку LUN.ua

**Виконав:** Антончук Іван
**Група:** ІПС-21

---

## 1. Мета роботи
Набуття практичних навичок автоматизації тестування користувацького інтерфейсу (UI) веб-застосунків за допомогою Selenium WebDriver та pytest, а також ознайомлення з принципами побудови архітектури автотестів (Page Object Model).

## 2. Стек технологій
* **Мова програмування:** Python 3.14
* **Фреймворк для тестування:** pytest
* **Інструмент автоматизації:** Selenium WebDriver
* **Маскування автоматизації:** undetected-chromedriver (для обходу захисту Cloudflare)
* **Архітектурний патерн:** Page Object Model (POM)

## 3. Структура проєкту
Проєкт побудований за принципом розділення логіки сторінок та самих тестів:
* `pages/base_page.py` — базовий клас із методами очікування, JavaScript-кліків та вводу тексту.
* `pages/home_page.py` — логіка головної сторінки та пошуку.
* `pages/results_page.py` — логіка взаємодії з каталогом (фільтри, валюта, мапа).
* `tests/` — папка з тестовими сценаріями.

## 4. Опис тестових сценаріїв

Для тестування було обрано 5 ключових сценаріїв взаємодії з сайтом LUN.ua:

1. **`test_search.py` (Пошук ЖК)**: Перевірка коректності роботи пошукового рядка. Ввід назви "Nordica Residence" та перевірка переходу на сторінку результатів за URL-адресою.
2. **`test_currency.py` (Зміна валюти)**: Перевірка функціонала перемикання валют. Тест відкриває бічне меню "Інше", обирає валюту "$" (USD) та перевіряє появу символу долара в цінах об'єктів.
3. **`test_language.py` (Зміна мови)**: Перевірка локалізації. Зміна мови інтерфейсу з української на англійську та перевірка наявності префікса `/en/` в URL.
4. **`test_filter.py` (Фільтрація)**: Перевірка роботи фільтрів у каталозі. Вибір фільтра "1-кімнатні" та перевірка оновлення параметрів у URL-адресі.
5. **`test_map.py` (Робота з картою)**: Перевірка навігації. Перехід до інтерактивної мапи новобудов та перевірка успішного завантаження картографічного розділу.
